# Mask vs No Mask Detection

This project implements a real-time face mask detection system using
CNN and transfer learning with MobileNetV2.

## Techniques
- Convolutional Neural Networks
- Transfer Learning
- OpenCV HaarCascade for face detection

## Dataset
Face Mask Detection dataset from Kaggle.

## Features
- Real-time webcam detection
- Bounding box around faces
- Green frame for mask / red frame for no mask
